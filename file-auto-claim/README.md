# FileAutoClaim

## Introduction

ABC Insurance Company is planning to build a solution to help its customers file Auto Insurance claim online.
This helps the users to quickly submit the claim in the occurrence of an unfortunate event like accident, loss, or theft of the vehicle.

You are expected to help ABC Company to achieve its goal by Creating a user interface to file a claim using Angular.
Please refer to the image ClaimUi.png in the root folder for the UI design.

## Technology Stack

1. Visual Studio Code.
2. Node version - 12+
3. Git client

## Points to be remembered while building the UI.

1. Create a claim form which accepts claim details and a submit button to trigger the api.
2. Once the claim is submiited. Display the claim details as shown in the image ClaimUi.png.
3. Do not the modify the form control name.
4. Do not modify the data-test-id.
5. once the development is completed run `ng e2e` to check the test results.
6. Do not modify the test cases.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.


## Running end-to-end tests

Run `ng e2e` to execute the end-to-end tests via [Protractor](http://www.protractortest.org/).
